Lightbot

Lightbot is a basic coding game that helps beginning coders understand basic coding. The game inculde premade programs and gives you the blocks nessesary to solve the problem. The blocks are very basic programs that move the robot up,down,left,right,and jumping. This game is very fun and educational.
